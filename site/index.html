Introduction

Les applications web progressives (PWA) sont des applications web qui offrent une expérience utilisateur similaire à celle des applications natives. Elles sont conçues pour fonctionner sur n'importe quel navigateur web et sur n'importe quel appareil, en utilisant les technologies web standard.

Les PWA peuvent être installées sur l'écran d'accueil de l'appareil de l'utilisateur, sans passer par un magasin d'applications (ex: App Store, Google Play). Elles peuvent fonctionner hors ligne et envoyer des notifications push.

Phase 1: Construction d'une application web avec cadriciel (React)

React est un librairie JavaScript (pas un framework) open-source utilisée pour développer des interfaces utilisateur réactives, principalement pour des applications web. Elle permet de créer des composants réutilisables et dynamiques qui se mettent à jour en fonction des changements de données.
